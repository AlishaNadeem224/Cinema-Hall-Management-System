# 🎬 Cinema Hall Management System

## 📌 Overview

The **Cinema Hall Management System** is a software application designed to manage a multi-screen cinema along with an integrated food court. It enables users to book movie tickets in advance, select seats, order food, and cancel tickets (only before the show begins). The system also provides administrative controls for managing movies, shows, screens, and food items efficiently.

---

## 🚀 Features

### 👤 User Features

* User Registration & Login
* View Movies and Show Timings
* Multi-screen selection
* Real-time seat availability
* Advance ticket booking
* Food court ordering with ticket
* Booking history tracking
* Ticket cancellation (allowed only before show time)

### 🛠️ Admin Features

* Add/Remove Movies
* Manage Screens and Seat Layout
* Schedule Shows
* Manage Food Menu
* View Bookings and Orders
* Generate Reports

### 💳 Payment Features

* Secure payment processing
* Automatic bill calculation (tickets + food)
* Refund on valid cancellation

---

## 🧩 Modules

* **User Module** – Handles user interaction and bookings
* **Admin Module** – Manages movies, screens, and reports
* **Booking Module** – Handles seat allocation and reservations
* **Food Court Module** – Manages food ordering system
* **Screen Management Module** – Handles multiple screens and shows
* **Payment Module** – Processes payments and refunds

---

## 🔄 System Flow

1. User registers/logs in
2. Browses movies and selects show
3. Chooses screen and seats
4. Optionally orders food
5. Makes payment
6. Booking is confirmed
7. User can cancel ticket **only before show time**

---

## ❌ Cancellation Policy

* Tickets can be cancelled **only if the show has not started**
* Refund is processed for valid cancellations
* No cancellation allowed after show start time
